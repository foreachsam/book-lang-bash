bash-doc 套件探索

下載套件

下載「bash-doc」這個套件。

$ apt-get download bash-doc

解開套件

解開「bash-doc_4.3-7ubuntu1.5_all.deb」

$ dpkg -x bash-doc_4.3-7ubuntu1.5_all.deb bash-doc

觀看資料夾結構

$ tree bash-doc
